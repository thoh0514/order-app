# Order App - Frontend

React + Vite 기반 프론트엔드 애플리케이션입니다.

## 개발 환경 설정

### 의존성 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

개발 서버는 기본적으로 `http://localhost:3000`에서 실행됩니다.

### 프로덕션 빌드
```bash
npm run build
```

### 빌드 미리보기
```bash
npm run preview
```

## 기술 스택

- **React 18**: UI 라이브러리
- **Vite**: 빌드 도구 및 개발 서버
- **Vanilla JavaScript**: 추가 JavaScript 라이브러리 없이 순수 JavaScript 사용
